# Base de prompts RS7138 Distanciel

> **Le bon prompt, en un clic.**

Cette application centralise et rend instantanément copiables les 38 prompts réels de la formation IA (RS7138). Elle permet aux stagiaires de trouver et copier n'importe quel prompt en moins de 10 secondes via des filtres combinables.

## 🚀 Fonctionnalités

- **Filtrage multicritères** : Filtrez par module, par paragraphe (§) ou par outil IA (ChatGPT, Gemini, Claude, etc.).
- **Recherche instantanée** : Recherchez par mot-clé dans le titre ou le contenu des prompts.
- **Copie en 1 clic** : Bouton de copie avec feedback visuel immédiat.
- **Responsive Design** : Interface premium optimisée pour mobile et desktop.
- **Zéro Configuration** : Chargement automatique des données depuis un fichier CSV.

## 🛠️ Stack Technique

- **Frontend** : React 18 + TypeScript
- **Tooling** : Vite
- **Styling** : Vanilla CSS (Modern UI, Glassmorphism)
- **Data** : Papa Parse (CSV Parsing)
- **Icons** : Lucide React

## 📦 Installation et Lancement

1. **Installer les dépendances** :
   ```bash
   npm install
   ```

2. **Lancer en mode développement** :
   ```bash
   npm run dev
   ```

3. **Build pour la production** :
   ```bash
   npm run build
   ```
